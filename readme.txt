== Installation ==
1. Unzip smart-scroll-to-top-lite.zip
1. Upload all the files to the /wp-content/plugins/smart-scroll-to-top-lite
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. For customizing the plugin's settings, click on Add New Settings, Selected Settings or Activated Settings in Wordpress left admin menu.


== Frequently Asked Questions ==
= What does this plugin do? =
This plugin helps you create, customize and build scroll to Top buttons for your site on your own.

= Do I have to make a Smart scroll button to start using the Plugin? =
Yes, creating a Scroll to top button is easy, and requires minimal knowledge. You are also familiarized with our plugins interface this way.

= What isn't this plugin? = 
This plugin isn't like other scroll to top plugins, where you are given many settings to fill up. We provide the users with apt and straightforward list of settings that requires little changes based on user's preferences that any naive user can configure. A predefined set of templates are given for buttons to choose from, which makes choosing between different scroll to top button layouts even easier.
